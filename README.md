# Word Builder

Word Builder is a small JavaScript project designed to practice phonics and
early reading skills through building simple CVC words and word families.

## Project Goals
- Practice JavaScript fundamentals (strings, arrays, loops)
- Generate and group CVC words
- Support phonics learning and blending skills
- Build the project gradually with small daily updates

## Tech Used
- HTML
- CSS
- JavaScript

## Status
This project is a work in progress and is updated incrementally as part of
daily coding practice.
